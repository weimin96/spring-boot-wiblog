## 技术栈

spring boot 、spring-mvc 、mybatis-plus 、mysql、redis 、elasticsearch、thymeleaf、vue、element-ui

## 环境搭建

1. 运行`myblog.sql`导入数据库
2. 配置mysql数据库账号密码、redis账号密码、服务器ip地址
3. -spring.profiles.active=dev/prod选择环境启动
